# thai-junior-pro-public

Public hosting for the **Thai Junior Pro** app's legal pages and installation guides (served via GitHub Pages).

The app source lives in the private `thai-junior-pro` repository; this repo only holds publicly-linkable documents.

**Live site:** https://thai-junior-pro.github.io/thai-junior-pro-public/

## Pages

- **Home / index:** [`index.html`](./index.html) — https://thai-junior-pro.github.io/thai-junior-pro-public/index.html
- **Android install guide:** [`install-android.html`](./install-android.html) — install via Google Play (Google Group testing track), EN + Thai side-by-side.
- **iOS install guide:** [`install-ios.html`](./install-ios.html) — install via TestFlight, EN + Thai side-by-side.
- **Privacy Policy:** [`privacy-policy.html`](./privacy-policy.html) — linked from the Google Play Console.

Screenshots for the guides live in [`images/`](./images).
