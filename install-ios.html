Installation Guide — iOS

Thai Junior Pro · คู่มือการติดตั้งสำหรับ iPhone / iPad

🤖 Using an Android phone? · ใช้มือถือ Android? → Android installation guide / คู่มือสำหรับ Android

1

Download TestFlight · ดาวน์โหลด TestFlight

Get TestFlight · ดาวน์โหลด TestFlight

Open the link above and tap Get / Install to download TestFlight from the App Store — Apple's free app for testing beta versions of apps.

เปิดลิงก์ด้านบนแล้วแตะ Get / Install เพื่อดาวน์โหลด TestFlight จาก App Store ซึ่งเป็นแอปฟรีของ Apple สำหรับทดสอบแอปเวอร์ชันเบต้า

⚠️ TestFlight is NOT our app — once it is installed, please proceed to the next step.
⚠️ TestFlight ไม่ใช่แอปของเรา — เมื่อติดตั้งเสร็จแล้ว กรุณาไปยังขั้นตอนถัดไป
2

Install the app · ติดตั้งแอป

Open the invite link · เปิดลิงก์คำเชิญ

Open the invite link above. This opens TestFlight on the "Thai Junior Pro" install screen — tap Install (or Update), then open the app from your Home Screen.

เปิดลิงก์คำเชิญด้านบน ระบบจะเปิด TestFlight ที่หน้าติดตั้ง "Thai Junior Pro" ให้แตะ Install (หรือ Update) แล้วเปิดแอปจากหน้าจอหลัก

✓

All done! · เสร็จเรียบร้อย!

If the install worked, opening the app shows the screen below — the player profile form. You're ready to use Thai Junior Pro!

หากติดตั้งสำเร็จ เมื่อเปิดแอปจะแสดงหน้าจอด้านล่าง ซึ่งเป็นแบบฟอร์มข้อมูลนักเตะ พร้อมใช้งาน Thai Junior Pro ได้เลย!
