Installation Guide — iOS

Thai Junior Pro · คู่มือการติดตั้งสำหรับ iPhone / iPad

1

Download TestFlight · ดาวน์โหลด TestFlight

Open the link below and tap Get / Install to download TestFlight from the App Store. TestFlight is Apple's free app that lets you help developers test beta versions of apps before they are released.

เปิดลิงก์ด้านล่างแล้วแตะ Get / Install เพื่อดาวน์โหลด TestFlight จาก App Store โดย TestFlight คือแอปฟรีของ Apple ที่ให้คุณช่วยนักพัฒนาทดสอบแอปเวอร์ชันเบต้าก่อนปล่อยจริง

⚠️ TestFlight is NOT our app — once it is installed, please proceed to the next step.
⚠️ TestFlight ไม่ใช่แอปของเรา — เมื่อติดตั้งเสร็จแล้ว กรุณาไปยังขั้นตอนถัดไป
Get TestFlight · ดาวน์โหลด TestFlight
2

Join the beta · เข้าร่วมการทดสอบ

Open the invite link below. It opens a TestFlight page in your browser — tap "Open" at the top right (highlighted in red below) to launch TestFlight and accept the "Thai Junior Pro" beta invitation.

เปิดลิงก์คำเชิญด้านล่าง ระบบจะเปิดหน้า TestFlight ในเบราว์เซอร์ ให้แตะ "Open" ที่มุมขวาบน (ไฮไลต์สีแดงตามภาพ) เพื่อเปิดแอป TestFlight และตอบรับคำเชิญทดสอบ "Thai Junior Pro"

Open the beta invite · เปิดลิงก์คำเชิญ
3

Install the app · ติดตั้งแอป

In TestFlight, on the "Thai Junior Pro" page, tap Install (or Update). Once installed, open the app from your Home Screen and you're ready to test.

ในแอป TestFlight ที่หน้า "Thai Junior Pro" ให้แตะ Install (หรือ Update) เมื่อติดตั้งเสร็จ ให้เปิดแอปจากหน้าจอหลัก แล้วเริ่มทดสอบได้เลย

✓

All done! · เสร็จเรียบร้อย!

If everything was installed successfully, opening the app should show the screen below — the player profile form. You're ready to start using Thai Junior Pro!

หากติดตั้งสำเร็จ เมื่อเปิดแอปจะแสดงหน้าจอด้านล่าง ซึ่งเป็นแบบฟอร์มข้อมูลนักเตะ พร้อมเริ่มใช้งาน Thai Junior Pro ได้เลย!
